# 프로젝트 문서

## 과목 관리

### subjects 테이블
- `subjects` 테이블에 과목 목록이 저장되어 있음
- 현재 "통합사회 1", "통합사회 2"가 DB에 존재하지만, 클라이언트에서는 사용하지 않음
- 통합사회 과목은 `lib/utils/subjectUtils.ts`의 `TONGHAP_SUBJECTS` 상수로 관리
  - id: "통합사회_1", "통합사회_2" (저장용)
  - label: "통합사회 1", "통합사회 2" (표시용)

### 수능 연도별 과목 필터링
- **2026학년도**: 통합사회 제외, 기존 과목만 선택 가능
- **2027학년도 이후**: 통합사회 1, 통합사회 2만 선택 가능
- 유틸 함수: `isAvailableSubject()`, `filterAvailableSubjects()` 참고
