# 봉투모의고사 MD → problem_id 변환 규칙

## MD 행 형식
```
{번호}. {년도}년 {학년} {월}월 모의고사 {과목} {문번}번({난이도}) {정답}
```
- 과목이 없는 경우 통합사회로 간주 (고1 문제)

## problem_id 형식
```
{과목}_{학년}_{년도}_{월}_{시험유형}_{번호}_문제
```

## 과목명 변환 (MD 표기 → problem_id)
| MD 표기 | problem_id |
|---------|-----------|
| 사회문화 | 사회문화 |
| 생활과 윤리 | 생활과윤리 |
| 윤리와 사상 | 윤리와사상 |
| 세계지리 | 세계지리 |
| 한국지리 | 한국지리 |
| 정치와법 | 정치와법 |
| 경제 | 경제 |
| 동아시아사 | 동아시아사 |
| 세계사 | 세계사 |
| (과목 없음, 고1) | 통합사회 |

|----|---------|

## 변환 예시
- `2025년 고2 3월 모의고사 사회문화 3번(중) 4번` → `사회문화_고2_2025_03_학평_3_문제`
- `2024년 고1 10월 모의고사 1번(하) 3번` → `통합사회_고1_2024_10_학평_1_문제`
- `2025년 고3 9월 모의고사 경제 5번(상) 3번` → `경제_고3_2025_09_모평_5_문제`

## 검증
- 변환된 problem_id가 `accuracy_rate` 테이블에 존재하는지 확인 필수
- 존재하지 않으면 오류 출력 후 스킵
