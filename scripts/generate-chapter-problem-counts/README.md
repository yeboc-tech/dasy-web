# 단원별 문제 개수 생성 스크립트

과목별, 단원별 문제 개수를 집계하여 JSON 파일로 저장하고 ssot 테이블에 업로드합니다.

## 실행 방법

```bash
npx tsx scripts/generate-chapter-problem-counts
```

## 데이터 소스

- **테이블**: `problem_tags`
- **필터**: `type LIKE '단원_%'`
- **필드**:
  - `problem_id`: 문제 ID (연도 추출용)
  - `type`: 태그 타입 (과목 추출용)
  - `tag_ids`: 단원 ID 배열 (마지막 요소 = 가장 구체적인 단원)

## 출력

### 1. 로컬 JSON 파일
- **경로**: `output/chapter-problem-counts.json`

### 2. ssot 테이블
- **key**: `단원별_문제_개수`

## 데이터 구조

```json
{
  "metadata": {
    "generatedAt": "2026-02-09T00:39:00.000Z",
    "currentYear": 2026,
    "recent3Range": "2024 ~ 2026",
    "recent5Range": "2022 ~ 2026"
  },
  "counts": {
    "경제": {
      "1-1": {
        "recent3": 5,
        "recent5": 15,
        "total": 29
      },
      "1-2": { ... }
    },
    "사회문화": { ... },
    "세계지리": { ... }
  }
}
```

## 필드 설명

| 필드 | 설명 |
|------|------|
| `recent3` | 최근 3개년 문제 수 (현재 연도 - 2 ~ 현재 연도) |
| `recent5` | 최근 5개년 문제 수 (현재 연도 - 4 ~ 현재 연도) |
| `total` | 전체 문제 수 |
